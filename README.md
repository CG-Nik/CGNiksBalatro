A Balatro mod that adds a variety of random content that I felt like adding, with intention for the content added to be balanced.

Thanks to the people who help modders in the Balatro discord.
Special thanks to N' for making Vanilla Remade, which is an incredibly useful resource for making Balatro mods, and La Ginger for creating a very useful consumable template.

I'm not affiliated with Lua, nor did I design the original Lua logo. The original designer is Alexandre Nakonechnyj. This mod includes several unofficial sprites that include pixel art representations of the Lua logo. You can find the official site for Lua at https://www.lua.org/.
