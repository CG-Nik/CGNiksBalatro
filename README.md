A Balatro mod that adds a variety of random content that I felt like adding, with intention for the content added to be balanced.

Thanks to the people who help modders in the Balatro discord.
Special thanks to N' for making Vanilla Remade, which is an incredibly useful resource for making Balatro mods.

I, obviously, didn't design the original Lua logo, and this mod does include a pixel art version of it. The original designer is Alexandre Nakonechnyj. You can find the official site for Lua at https://www.lua.org/.
